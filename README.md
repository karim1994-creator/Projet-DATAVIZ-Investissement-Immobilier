# Projet DATAVIZ – Investissement Immobilier en France

## Description

Projet de Data Visualisation visant à identifier les villes françaises les plus attractives pour un investissement immobilier locatif.

L'analyse repose sur les données de transactions immobilières (DVF) et les données de loyers afin d'étudier les prix, les loyers et les rendements locatifs à travers la France.

## Objectifs

* Identifier les villes les plus rentables.
* Comparer les marchés immobiliers français.
* Analyser la relation entre prix, loyers et rendement.
* Aider à la prise de décision pour un investissement immobilier.

## Technologies utilisées

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Tableau

## Fichiers du projet

* `Notebook_final.ipynb` : préparation, analyse et tests statistiques.
* `Projet DATAVIZ.pptx` : présentation du projet.
* `Rapport_Tableau_Rendement_Inv_Immobilier.twb` : dashboard Tableau.
* `investissement_immobilier_france.csv` : données finales utilisées pour l'analyse.

## Principaux résultats

* Les villes aux prix modérés offrent souvent les meilleurs rendements.
* Le prix au m² influence fortement la rentabilité.
* Les rendements locatifs varient selon les régions.
